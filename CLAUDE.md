# Claude Code Project Configuration

This file is read automatically at the start of every Claude Code session.

## Session Startup

Before doing anything else, read these files in order:
1. `documents/ai-usage/policies/HOMEWORK-MANIFESTO.md` (code authorship rules - always active)
2. `documents/docs/PROJECT-CONTEXT.md` (development rules, testing, current state)
3. `documents/docs/FILE-ORGANIZATION.md` (mandatory file placement rules)

## Global Rules

- Do not use emojis in any output, code, comments, or documentation

- Always propose file locations before creating files
- Follow the file organization rules in `documents/docs/FILE-ORGANIZATION.md`
- `documents/ai-usage/policies/HOMEWORK-MANIFESTO.md` code authorship rules are NON-NEGOTIABLE
